Navbar

Bootstrap offers a navbar component that is responsive and fluid by default. The navbar can be adjusted in any way you want, but by default, the links hide in a hamburger menu for tablet and mobile. The default template includes drop down menus, but I am personally not a big fan of them, unless they are absolutely necessary to accomodate a lot of content. I also made the navbar stick to the top, so a user can easily move to different pages.

Banner

I created a simple banner with a Bootstrap preset color and wrapped the banner content in the container. I used a bootstrap button component in primary style to use it as a call-to-action. Buttons come in different styles that serve their own semantic values, and there are also dark, light, and link styles.

Some Bootstrap components require additional margins, so I added them where needed.

Warning! Boodstrap will make your life easier!

Alert Component

I used an alert component to bring a user's attention to important information on the page. The color yellow is universally conveyed as a warning color, where red is considered danger.

Cards Component

Card are a convenient component to display a gallery and provide previews. Like most components in Bootstrap, cards use flexbox. Bootstrap uses a 12 column grid, so you can easily adjust how many cards you want to be displayed in a row depending on the screen size. On mobile, I used 1 column to display cards, 2 columns for tablet, and 3 on desktop.

I customized the cards to add rounded corners and a shadow. The shadow styles are also a part of Bootstrap presets, so they don't require additional CSS, which is super convenient. I also chose the outline style for buttons inside the cards to create visual hierarchy with the main call-to-action buttons.

Form Component

I'm personally not a big fan of forms and find them annoying to customize. Bootstrap offers a variety of forms, but I chose a standard contact form. It is full width by default, and since I put all content in a container, it always looks good regardless of the screen size. However, I noticed that if you use textarea, Bootstrap doesn't restrict resizing, so you need additional CSS to change that.

Footer Content

This is a Bootstrap footer template that includes a header, a paragraph, and an unordered list of links used for a site map. It also has a separate div for the copyright line. For mobile screens footer content and site map are in 1 column, in tablet they take up equal space side by side and the links are aligned to the end. The footer content is also in a container to match the page content.
